## ⚡ Highlighted Project: PhibERP (ERPNext SaaS on Kubernetes)

I designed and deployed **PhibERP**—an enterprise-ready ERPNext SaaS environment running on a multi-node Kubernetes cluster.

*   🔗 **Repository:** [phib256/erpnext-kubernetes](https://github.com/phib256/erpnext-kubernetes)
*   🛡️ **Security:** Connected to the edge via **Cloudflare Zero Trust Tunnels**, bypassing the need for public IP addresses or firewall exposures.
*   💾 **Distributed Database**: Dedicated, isolated database clusters utilizing MariaDB StatefulSets and dynamic persistent block storage.
*   ⚡ **Smart Storage & Scheduling:** Overcame multi-node NFS mounting bottlenecks by implementing Kubernetes **volume-affinity scheduling**, binding frontend and background worker queues to share local node directories at native SSD speeds.
